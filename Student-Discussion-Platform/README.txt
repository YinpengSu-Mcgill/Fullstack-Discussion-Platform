# Full Stack Discussion Platform

## 📌 Overview
A web-based discussion platform that allows users to:
- Create and join boards
- Post messages
- Manage discussion channels

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Server: Apache (XAMPP)

## 🚀 How to Run

1. Install XAMPP
2. Place project in:
   /xampp/htdocs/

3. Start:
   - Apache
   - MySQL (if needed)

4. Open browser:
   http://localhost/index.php

## 📂 Project Structure
- /api → backend APIs
- /controllers → business logic
- /assets → static resources
- /dashboard → UI
